### 初始化  
- git init  

### 查看状态  
- git status  

### 添加到暂存区  
- git add .  

### 暂存区file恢复工作区file  
- git checkout -- filename  

### 删除文件  
- git rm filename  

### 提交到版本库  
- git commit -m "comment"  

### 远程库  
- git add remote origin git@github.com:name/project.git  
- git push -u origin master  
- git clone git@github.com:name/project.git
